1. Logo & Brand Mark

Primary Logo

The PaperBook logo consists of a stylized "P" lettermark with an orange saffron triangle accent, representing growth and progress in education.

Logo Anatomy

ElementDescriptionColor
BackgroundRounded rectangle (8px radius)Purple gradient
P LettermarkWhite stylized letter P#FFFFFF
P Inner BowlCutout showing gradientBackground gradient
Triangle AccentAcute triangle pointing up#FF9933 (Saffron)

Triangle Specifications

The triangle accent is an acute isoceles triangle with:

PointCoordinatesDescription
Bottom-left(19, 25)Aligned with P's bottom
Bottom-right(27, 25)8 units from left point
Apex(23, 18)Centered, pointing straight up

Logo Variations

Light Background

Dark Background

On Brand Gradient

Minimum Clear Space

Maintain clear space equal to the height of the "P" lettermark around all sides of the logo. Never place other elements within this zone.

10. Illustration Guidelines

Illustrations are custom SVG graphics that represent dashboard interfaces, features, and concepts.

Style Characteristics

  • Flat design with subtle shadows
  • Dashboard/UI mockup style
  • Rounded corners (4-16px based on element size)
  • Module accent colors for key elements
  • White backgrounds with light gray (#f8fafc) sidebars
  • Window controls (red, yellow, green circles)

Illustration Elements

ElementStyle
Main FrameWhite fill, 2px #e2e8f0 stroke, 16px radius
Header Bar#f8fafc fill, window controls
Sidebar#f5f3ff (module light) fill
Data CardsRounded rects with module colors
ChartsSimple bar/line charts with gradients
Icons2px stroke, module primary color
Text PlaceholdersRounded rectangles, varying widths

Color Usage in Illustrations

PurposeColors
Backgrounds#ffffff, #f8fafc, #f5f3ff
Borders#e2e8f0

Success States#22c55e, #d1fae5
Highlight/ActiveModule primary with 0.1-0.3 opacity

Illustration Animation

Hero illustrations use the floatGentle animation (6s ease-in-out infinite) for subtle movement. Feature illustrations fade and slide in on scroll reveal.
